# 飲料統計器 with Google Sheets API
一、說明
1. 簡介
	*  公司在訂購下午茶時是使用 google sheets 表單來記錄，為了方便統計金額、杯數與人數，寫了一個小程式來幫助統計。
2. 開發語言
	* C#
3. 套件
	* Google - Sheet API
